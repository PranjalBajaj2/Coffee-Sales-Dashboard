# Coffee-Sales-Dashboard

This project analyzes coffee shop sales data to understand sales performance, customer payment behavior, and product-wise sales trends. The Power BI dashboard transforms raw transactional data into clear, actionable business insights that can help café owners and managers make informed decisions.
The focus is on sales trends, top-selling coffee products, and payment method analysis using interactive visuals.

# Objectives

Analyze total coffee sales 

Identify top-performing coffee products

Track sales trends over time

Present insights through an interactive Power BI dashboard

# Dataset Description

The Coffee Sales Dataset includes:

coffee_name – Type of coffee sold

money – Sales amount per transaction

date / datetime – Date and time of purchase

cash_type – Payment method (Cash or Card)

card – Indicator for card transactions

# Tools & Technologies Used

Power BI Desktop

Power Query for data cleaning and transformation

DAX for calculated measures

Data Visualization & Business Intelligence

# Key Insights

Certain coffee types generate significantly higher revenue

Sales fluctuate over time, highlighting peak business periods

Card payments are widely used compared to cash, indicating a shift toward digital transactions
